# React 시작하기
프로젝트 만들기 : npx create-react-app [프로젝트 이름]
프로젝트 시작하기 : npm start
vite react 프로젝트 만들기 : npm create vite@latest [프로젝트 이름]
vite react 프로젝트 시작하기 : npm run dev
